Chúng tôi cung cấp cho bạn các công cụ để tự động: Khai phá dữ liệu, gợi ý tiền xử lý, chạy và so sánh các mô hình.
Từ đó các bạn sẽ hiểu hơn về bộ dữ liệu, có thể tự tạo ra một base-line riêng.
Hiện tại Website hỗ trợ 3 loại định dạng file dataset: csv, excel, json.

Các bước thực hiện:

  1. Tải file dữ liệu: chọn file dataset mình muốn chạy tại đây
  2. Chạy Khai phá dữ liệu:
    • Phần tổng quan: Hiển thị thông tin tổng quát của dataset
    • Phần Cảnh báo: Hiển thị những phần lỗi mà dataset đang mắc phải
    • Phần Đặc trưng: Hiển thị sự tương quan cũng như tương tác giữa các đặc trưng
    • Phần Gợi ý: Gợi ý cách để tiền xử lý bộ dữ liệu hiện tại
  3. Mô hình hóa : Chọn target và features để huấn luyện, sau đó hiển thị những mô hình tốt nhất
